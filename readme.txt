An easy way to connect products from multiple marketplaces to your WooCommerce.

== Description ==

ProChains is the best tool for Dropship and Affiliates, build your own store easily.

You can get products and sell them on WooCommerce just from the product url in marketplace. ProChains will automatically extract the product data for you.

For now, our service is integrated with Shopee, Tokopedia and Bukalapak. In the future, our services will be integrated with several other leading marketplaces in Indonesia.

== Installation ==

= Minimum Requirements =

* WooCommerce 6.4.1 or greater is recommend
* PHP 7.4 or greater is recommended
* MySQL 5.7 or greater is recommended

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of ProChains, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “ProChains”, then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the ProChains plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
